# 快速开始指南

欢迎使用低代码引擎！本指南将帮助您在 5 分钟内创建第一个页面。

## 前置准备

- 已完成 [环境搭建](../development/deployment/development.md)
- 拥有系统账号

## 创建第一个页面

### 步骤 1: 登录系统

访问 [http://localhost:3000](http://localhost:3000) 并使用您的账号登录。

### 步骤 2: 创建新页面

1. 点击仪表盘中的"新建页面"按钮
2. 输入页面名称，如"用户管理"
3. 点击"创建"

### 步骤 3: 添加组件

1. 从左侧组件库拖拽一个 Container 组件到画布
2. 在 Container 中添加一个 Text 组件作为标题
3. 再拖拽一个 Table 组件到 Container 中

### 步骤 4: 配置数据

1. 选中 Table 组件
2. 在右侧属性面板中配置数据源
3. 设置列标题和字段映射

### 步骤 5: 预览和保存

1. 点击"预览"按钮查看效果
2. 确认无误后点击"保存"
